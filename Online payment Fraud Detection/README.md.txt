# Online Payment Fraud Detection using Machine Learning

This project detects fraudulent online payment transactions using machine learning techniques.

Technologies Used:
- Python
- Pandas
- Scikit-learn
- Machine Learning

Algorithm:
Random Forest Classifier

Project Description:
The system analyzes transaction data such as amount and account balance changes to classify transactions as fraudulent or legitimate.
